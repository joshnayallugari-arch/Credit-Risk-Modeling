Description:
Credit Risk Modeling is a machine learning project used to predict whether a customer is likely to repay a loan or default on it. This project analyzes customer financial and loan-related data such as loan amount, payment history, rental details, and transaction patterns to identify risky borrowers.

The project involves data preprocessing, feature selection, correlation analysis, model training, and performance evaluation using machine learning algorithms like Logistic Regression, K-Nearest Neighbors, Decision Tree, Random Forest, and Naive Bayes.

The main goal of this project is to help financial institutions reduce loan default risk by making accurate credit approval decisions based on customer risk prediction.
